# nyc-subway-ridership-analysis
Analyzing the NYC MTA weekday data from 2018 - 2023 to enhance the knowledge and understanding of subway ridership before, during, and after the covid pandemic. 

## Tools Used
- SQLite
- Tableau
- Excel
- GitHub
